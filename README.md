# Презентация: Индукция и Дедукция

Интерактивная веб-презентация о математической индукции и дедукции для 11 класса.

## Технологии

- Next.js 14
- React 18
- TypeScript
- Tailwind CSS
- Framer Motion (анимации)
- KaTeX (математические формулы)

## Запуск локально

```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev

# Сборка для продакшена
npm run build

# Запуск продакшен-версии
npm start
```

Презентация будет доступна по адресу [http://localhost:3000](http://localhost:3000)

## Управление

- **Стрелки влево/вправо** - навигация между слайдами
- **Кнопки внизу** - переход к предыдущему/следующему слайду

## Деплой на Vercel

1. Загрузите проект на GitHub
2. Импортируйте проект в Vercel
3. Деплой произойдет автоматически

Или через Vercel CLI:

```bash
npm i -g vercel
vercel
```
